Embedded Systems Final Project

Smart Autonomous Monitoring & Control System

This project combines environmental sensing, dynamic output control, user interaction, and real-time telemetry display into a single, cohesive system. It is designed as a scalable prototype for real-world applications in smart homes, industrial monitoring, and safety automation.

Project Objectives

The goal was to integrate multiple fundamental embedded systems concepts into one robust architecture.

Automation

Continuously monitor the environment and execute logic-based responses dynamically.

Safety Thresholds

Detect hazardous conditions such as gas leaks, extreme temperatures, or proximity breaches.

User Override

Provide physical interfaces allowing the user to seamlessly switch between operational states.

System Sub-Modules

The architecture is divided into six distinct operational requirements.

1

Basic Output Control

Discrete LEDs visually broadcast the overarching system state.

Green = SafeYellow = WarnRed = Alert

2

Variable Actuation

A PWM-driven LED linearly adjusts its brightness inversely proportional to ambient light (LDR).

3

State Management

Hardware interrupts/polling via tactile buttons to toggle Auto, Manual, and Emergency modes.

4

Audible Alerts

A piezoelectric buzzer is triggered with varying frequencies based on the severity of the hazard.

5

Telemetry Display

A 16x2 LCD renders real-time data including temperature, luminosity, distance, and current mode.

6

Environmental Sensing

An array of analog and digital sensors (LDR, DHT11, MQ-2, Ultrasonic) gather real-world data.

Bill of Materials (Hardware)

The core physical components utilized to prototype this system.

Microcontroller & Sensors

  • Arduino Uno R3 (ATmega328P)
  • DHT11 Digital Temperature & Humidity Sensor
  • LDR (Photoresistor) Light Sensor
  • MQ-2 Gas / Smoke Sensor
  • HC-SR04 Ultrasonic Distance Sensor

Actuators & Peripherals

  • 16x2 Character LCD Display (Hitachi HD44780 compatible)
  • Red, Yellow, and Green 5mm Indicator LEDs
  • White LED (PWM controlled)
  • Active Piezo Buzzer
  • Push Buttons, Solderless Breadboard, Jumper Wires, Resistors (220Ω, 10kΩ)
